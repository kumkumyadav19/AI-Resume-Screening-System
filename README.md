# AI Resume Screening & ATS System

## Overview
This project is an AI-powered Applicant Tracking System (ATS) that automates resume screening, candidate ranking, and job matching using NLP and machine learning.

## Features
- Resume parsing (PDF)
- Skill extraction
- Multi-job matching
- Resume classification (ML model)
- Semantic similarity using Sentence Transformers
- Skill gap analysis
- ATS-based scoring

## Tech Stack
- Python
- Sentence Transformers
- Scikit-learn
- Google Sheets API
- Google Drive API

## Pipeline
Resume → Parsing → Preprocessing → Feature Extraction → Classification → Matching → Ranking

```bash
pip install -r requirements.txt
